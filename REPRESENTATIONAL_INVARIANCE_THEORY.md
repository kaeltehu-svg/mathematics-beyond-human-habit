# Representational Invariance Theory

## Purpose

This note names and defines the broader disciplinary direction that may emerge from the multibase starting point.

It does not claim that the discipline already exists in finished form.

It states the hypothesis clearly:

**representation-change can be used as a structural measurement tool.**

The core question is not only how to represent an object.

The core question is:

**what remains invariant, and what changes, when the representation changes?**

---

## Working definition

**Representational Invariance Theory** is the systematic study of which properties of an object remain stable, which drift, and which collapse when the object is observed through multiple admissible representations.

Its aim is to separate:

- the object itself
- the encoding of the object
- the properties that survive representational change
- the properties that are introduced by a specific encoding

In compressed form:

**a property has stronger structural status if it survives changes of representation.**

---

## Why this is not just standard notation theory

Mathematics already knows that the same object can admit multiple representations.

For numbers, this includes different bases.  
For geometry, different coordinate systems.  
For logic, different symbolic encodings.  
For computation, different data formats or tokenizations.

That is not new.

What is potentially new is the methodological shift:

- not using multiple representations only as alternative descriptions
- but using them as probes of structural stability

This is the key distinction.

The question is no longer only:

**how can this object be written?**

The question becomes:

**which observed properties belong to the object, and which belong only to the chosen representation?**

---

## Central hypothesis

The central hypothesis is:

**variation across representations is not just nuisance; it is measurable signal.**

And more strongly:

**a property that persists across many representations has higher structural credibility than a property that appears only in one encoding.**

If this hypothesis can be formalized and tested, then representational variation becomes a method of structural discrimination.

That is the possible birth point of the discipline.

---

## Object of study

The true object of study is not a single notation.

It is the relation between:

1. an object
2. a family of admissible representations
3. the properties observed across that family
4. the invariants and drifts induced by representational change

So the disciplinary unit is not:

- one number in one base
- one formula in one notation
- one object in one encoding

The disciplinary unit is:

**an object across its representational orbit.**

---

## Minimal schema

At the most abstract level, the discipline studies:

- an object `x`
- a set of admissible representations `R(x)`
- a family of observed properties `P`
- the behavior of each property under changes of representation

The core classification problem is:

- **invariant properties**: stable across representation changes
- **drifting properties**: modified but not destroyed by representation changes
- **collapsing properties**: disappear when the representation changes
- **encoding artifacts**: properties that arise only inside one specific representation

This classification is more important than any single notation.

---

## What it measures

A mature form of the discipline would aim to measure quantities such as:

- representational stability
- representational drift
- compression variability
- persistence across encodings
- structural robustness under re-expression
- artifact sensitivity

These names are provisional.

The underlying point is not the vocabulary.

The point is that representational change is treated as something measurable, not merely translatable.

---

## Why the multibase case matters

The multibase case is the simplest clean entry point.

For integers, the object stays the same while the written form changes with the base.

This makes it possible to observe, without semantic ambiguity, that:

- the object is invariant
- the representation is not
- the descriptive cost depends on the representational system

This is why the multibase case matters.

It is not the full discipline.

It is the minimal visible case in which the discipline can begin.

---

## What was already known

It was already known that:

- the same number can be written in different bases
- the written form changes with the base
- digit-length changes with the base
- different representations can make the same object appear longer, shorter, simpler, or more complex

These are not discoveries of Representational Invariance Theory.

They are the raw material.

The possible novelty begins only when these known facts are used as instruments of structural measurement.

---

## What may be new

What may be new is the move from:

**representation as notation**

to:

**representation as test**

And then further:

**representation-change as a method for detecting structural invariance, drift, and artifact.**

This is the point at which a family of old facts may become a new discipline.

---

## Core questions

The discipline would ask questions such as:

- Which properties remain stable across multiple representations?
- Which properties are highly representation-dependent?
- Which apparent regularities survive re-expression?
- Which regularities disappear once the encoding changes?
- Can representational drift be measured?
- Can structural robustness be ranked across objects or properties?
- Can encoding artifacts be separated from intrinsic structure?

These questions define the field more clearly than any slogan.

---

## What it is not

Representational Invariance Theory is not:

- a claim that all representations are equally useful
- a rejection of standard mathematics
- a replacement for proof
- a purely aesthetic preference for multiple views
- a claim that one must always use all representations

It is also not a completed theory.

At this stage, it is a disciplined research direction.

Its value depends on formal definitions, explicit metrics, and falsifiable results.

---

## Why this may matter beyond numbers

If valid, this framework is not limited to numeral bases.

The same logic can extend to:

- symbolic mathematics
- coordinate systems
- geometric encodings
- logical formalisms
- data representation
- tokenized language
- computational states
- model outputs under alternative encodings

So the multibase case should be treated as the first domain, not the final domain.

---

## Relation to OMNIABASE

OMNIABASE can be understood as the first concrete domain in which Representational Invariance Theory is explored.

Its role is specific:

- start from numbers
- vary the base
- observe what changes
- measure what persists

That makes OMNIABASE a domain-level entry point.

Representational Invariance Theory is the broader conceptual frame.

So the relation is:

- **OMNIABASE** = initial operational domain
- **Representational Invariance Theory** = broader disciplinary hypothesis

---

## Minimal claim

The minimal claim is modest but strong:

**an observed property should not automatically be treated as intrinsic to an object unless it survives representational change.**

This does not solve every problem.

But it introduces a stricter discipline of separation between structure and encoding.

That separation is the real point.

---

## Strong form

In stronger form, the thesis becomes:

**the structural credibility of a property increases with its persistence across admissible representations.**

This is not yet a theorem.

It is the central research hypothesis.

Its future depends on whether it can be made measurable and falsifiable.

---

## Practical research direction

A serious development of this discipline would require:

1. defining admissible representation families
2. defining property classes to be tracked
3. defining metrics for invariance, drift, and collapse
4. distinguishing intrinsic structure from encoding artifact
5. testing the framework on cases where representation clearly matters
6. extending from numbers to richer formal objects

Without these steps, the idea remains conceptual.

With them, it may become a real method.

---

## Compressed summary

**Representational Invariance Theory** studies what remains stable and what changes when the same object is observed through multiple admissible representations.

Its central claim is that cross-representation persistence is evidence of stronger structural status, while representation-dependent effects must be treated with caution.

The multibase case is the simplest starting point.

It is not the end of the theory.

It is the cleanest place to begin.
